# CloudCare-Monitor
A cloud-based system for tracking patient health data in real time. It collects vitals, analyzes them, and provides dashboards and alerts to support faster and smarter healthcare decisions.

## Features
Real-time health data collection and analysis

Dashboards for patients and doctors

Alerts for critical health conditions

Secure data storage and transfer

Scalable cloud architecture

## Tech Stack
AWS DynamoDB – Cloud database for storing patient health data

AWS Lambda – Serverless functions for processing data

Amazon API Gateway – Secure endpoints for communication

Flask (Python backend) – Connects frontend with cloud services

Pandas & Matplotlib – Data processing and visualization

SSL Encryption – Ensures secure data transfer

CSV Dataset – Simulated patient vitals for demo purposes
